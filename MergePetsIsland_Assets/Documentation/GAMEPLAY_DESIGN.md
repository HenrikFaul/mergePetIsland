# Gameplay Summary

Core loop: spawn items → merge identical pets/items → discover higher-level pets → complete orders → earn coins/gems → upgrade island buildings → unlock new biomes → repeat.

Monetization: rewarded ads for speedups and extra rewards, IAP coin/gem packs, remove ads, season pass, event offers.

Retention: daily rewards, pet collection, island expansion, time-limited events, orders, leaderboards and offline earnings.
