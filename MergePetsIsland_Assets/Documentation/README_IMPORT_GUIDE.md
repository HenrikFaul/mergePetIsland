# Merge Pets Island — Premium Unity Import Asset Pack

This package keeps the original folder structure and replaces placeholder-style entries with generated, usable standalone assets.

## Import
1. Copy `MergePetsIsland_Assets` into `Assets/` in Unity.
2. Set PNG Texture Type to Sprite (2D and UI).
3. For character/environment sprites, use Pixels Per Unit 100 or 128.
4. For UI assets, use Sprite Mesh Type: Full Rect.
5. Use JSON/CSV files in `Data/` for economy, pets, buildings and orders.

## Notes
- PNG assets are original generated vector/raster-style assets with transparent backgrounds where appropriate.
- Audio WAV files are simple original generated tones/loops. Replace with studio audio later if desired.
- Prefab/material/animation files are Unity-ready metadata placeholders, because full Unity serialization normally requires the Unity Editor.
